Quick Start A 5 minutes introduction

  1. Home
  2. Quick Start
You need to have Git and Erlang installed.
  1. First download paris, make it executable and add it in you PATH.
  2. Install templates by running : {% highlight sh %} paris install {% endhighlight %}
  3. Create a new paris' project using : {% highlight sh %} paris new my_project {% endhighlight %}
  4. Et voilà! {% highlight sh %} cd my_project make ./my_project start {% endhighlight %}
  5. Visit http://localhost:8080 with your favorite browser.
